Exercise 5

Enunciado:
Utilizando el ciclo que usted desee, crear un menú de ejecución infinita hasta que el usuario desee terminar dicho ciclo.

                Proceso Taller3_5
                    Definir option, select Como Entero;
                    Definir name, sayHi Como Caracter;
                    
                    select<-1;
                    
                    Mientras select = 1 Hacer
                        Escribir "Menú de usuario, ingrese un número segun la opcioón deseada";
                        Escribir "1. Captura nombre";
                        Escribir "2. Saludar persona";
                        Escribir "3. Salir del sistema";
                        Leer option;
                        
                        Segun option Hacer
                            1:
                                Escribir "Ingresa su nombre";
                                Leer name;
                                Limpiar Pantalla;
                            2:
                                Escribir "Saluda";
                                Leer sayHi;
                                Limpiar Pantalla;
                            3:
                                Escribir "Salir del sistema";
                                
                            De Otro Modo:
                                Escribir "La opción seleccionada no existe";
                                Escribir "Intentalo de nuevo";
                        FinSegun
                        
                        Escribir "";
                        Escribir "¿Desea regresar al menú?";
                        Escribir "1. Ingrese 1 para volver al menú";
                        Escribir "2. Ingrese un número diferente a 1 para salir del menú";
                        Leer select;
                        
                    FinMientras
                        
                FinProceso
