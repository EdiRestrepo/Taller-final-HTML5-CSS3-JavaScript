Enunciado:
La video tienda que presta sus servicios de alquiler de películas a los usuarios del barrio el Porvenir, requiere de una aplicación que permita registrar el alquiler de las películas que adquieren sus usuarios. Para cada usuario se debe permitir la opción de alquilar película, consultar películas disponibles y recibir película en la video tienda con la opción de realizar anotaciones sobre estas si se llegan a presentar daños u otra novedad sobre la película.

                        private String name;
                       private int option;
                       private int[] available;
                       private String[] movie;
                       private boolean flag = true;
                       private String[] annotations;
                    
                        public void Ejercicio4 () throws IOException {
                            available = new int[5];
                            movie = new String[5];
                            annotations = new String[5];
                            available[0]=0;
                            available[1]=0;
                            available[2]=0;
                            available[3]=0;
                            available[4]=0;
                            movie[0]="Red";
                            movie[1]="Sonic";
                            movie[2]="Sing";
                            movie[3]="Space Jam";
                            movie[4]="Raya";
                            System.out.println("APP video tienda alquiler de películas Porvenir");
                            askName();
                            name = captureDataString();
                            System.out.println("Hola "+name+", bienvenido a su video tienda Porvenir");
                            do {
                                menu();
                                option = captureDataInt();
                                flag = evaluateOption(option);
                            }while (flag == true);
                        }
                    
                        private void menu(){
                            System.out.println("Por favor, digite un número de acuerdo a lo que desea hacer");
                            System.out.println("1. Alquilar película");
                            System.out.println("2. Consultar películas disponbles");
                            System.out.println("3. Recibir película en la video tienda");
                            System.out.println("0. Salir de la App");
                        }
                    
                        private void askName(){
                            System.out.println("Por favor, digite su nombre de usuario");
                        }
                    
                        private String captureDataString(){
                            return input.nextLine().replace("\\n", "");
                        }
                    
                        private int captureDataInt(){
                            return Integer.parseInt(input.nextLine());
                        }
                    
                        private boolean evaluateOption(int option) throws IOException {
                            switch (option){
                                case 1:
                                    listMovies();
                                    if((available[4]==0) || (available[3]==0) || (available[2]==0) || (available[1]==0) || (available[0]==0)){
                                        System.out.println("Por favor, digite un número de a acuerdo a la película que deseas alquilar");
                                        option = captureDataInt();
                                        rentMovie(option);
                                    }
                                    return true;
                                case 2:
                                    listMovies();
                                    if((available[4]==0) || (available[3]==0) || (available[2]==0) || (available[1]==0) || (available[0]==0)){
                                        System.out.println("Digita, enter para volver al menu principal");
                                        System.in.read();
                                    }
                                    return true;
                                case 3:
                                    rentedFilms();
                                    if((available[4]==1) || (available[3]==1) || (available[2]==1) || (available[1]==1) || (available[0]==1)){
                                        System.out.println("Por favor, digite un número de a acuerdo a la película que vas a recibir");
                                        option = captureDataInt();
                                        returnFilm(option);
                                    }
                                    return true;
                                case 0:
                                    return false;
                                //break;
                                default:
                                    System.out.println("Opción incorrecta");
                                    System.out.println("Digita, enter para volver al menu principal");
                                    System.in.read();
                                    return true;
                            }
                        }
                    
                        private void listMovies() throws IOException {
                            if ((available[4]==0) || (available[3]==0) || (available[2]==0) || (available[1]==0) || (available[0]==0)){
                                System.out.println("Lista de peliculas disponibles");
                            }
                            if(available[0]==0){
                                System.out.println("1. "+movie[0]);
                            }
                            if(available[1]==0){
                                System.out.println("2. "+movie[1]);
                            }
                            if(available[2]==0){
                                System.out.println("3. "+movie[2]);
                            }

                                System.out.println("4. "+movie[3]);
                            }

                                System.out.println("5. "+movie[4]);
                            }

                                System.out.println("Lo sentimos, pero en el momento no tenemos peliculas disponibles");
                                System.out.println("Digita, enter para volver al menu principal");
                                System.in.read();
                            }
                        }
